# REST API «Шесть городов»

Сервис аренды жилья для путешественников, не желающих переплачивать за проживание в одном из шести популярных городов.

Проект предоставляет:

- REST API
- CLI для генерации и импорта тестовых данных
- интеграцию с фронтендом в директории `frontend`

Подробный запуск и список сценариев описаны в [Workflow.md](/home/sea/univers/2503307-six-cities-6/Workflow.md).
